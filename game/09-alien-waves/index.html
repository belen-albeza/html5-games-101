We need to use sprite pooling to handle the aliens, like we did with the bullets. We will also destroy them once they are no longer visible.

Aliens behaviour is to fall towards the bottom of the screen, at a random speed (both horizontally and vertically). There is a random number generator at game.rnd. We will use its between method, which returns a random integer within a range, to set the alien's speed.

We will be spawning aliens at a random x position. Every frame, there will be a certain probability that a new alien will be spawned. To simulate this, we can use the between random method again.

PlayState.update = function () {
    // ...
    if (this.game.rnd.between(0, 100) < 10) { // 10% chance
        Alien.spawn(...);
    }
};
