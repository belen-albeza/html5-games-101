Our code might be more clean if we create our own custom sprite classes so we can add methods to them, like shoot or explode. We can extend Phaser.Sprite to do that.

To extend (or inherit from) Phaser.Sprite, you can follow this pattern:

function MySprite(game, x, y) {
    // call Phaser.Sprite constructor
    Phaser.Sprite.call(this, game, x, y, 'image_key');
    // other init code goes here
}

// clone Phaser.Sprite prototype into MySprite's
MySprite.prototype = Object.create(Phaser.Sprite.prototype);
// restore MySprite constructor
MySprite.prototype.constructor = MySprite;
