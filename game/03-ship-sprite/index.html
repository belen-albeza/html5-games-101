Sprites are, roughly, images with behaviour and movement. Most of our game elements will be sprites: the player's ship, the enemies, the bullets, etc. In Phaser they are instances of Phaser.Sprite.

As with images, there is a factory method we can use to create sprites and add them to the game world.

game.add.sprite(0, 0, 'ship');

All sprites have an anchor point that acts at their origin of coordinates for rendering and other transformations like rotating. It is common to put this point at the center of the image. We can do this with sprite.anchor.setTo(0.5).

If we want to use Phaser's Arcade physics engine to handle our sprite's movement, etc. we need to enable it manually with game.physics.enable(sprite). The sprite then will have a body property which we can manipulate.

Uniform movement can be achieved by setting the sprite speed in body.velocity.
