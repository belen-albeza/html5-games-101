Music and sound effects in Phaser work in a similar way to images. We load the assets in the preload phase. Afterwards, we create instances of Phaser.Sound that will use those audio assets we have loaded.

game.load.audio('music', 'coolsong.mp3');

As with images, Phaser.Sound instances need to be added to the game world in order to be used.

let sfx = game.add.audio('explosion');
sfx.play();
