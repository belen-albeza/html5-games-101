To be able to read the keyboard and listen for events, we need instances of Phaser.Key. We can create these instances in several ways, the most handy one being the Phaser.Keyboard.addKeys method. Example:

let keys = game.input.keyboard.addKeys({
    left: Phaser.KeyCode.LEFT,
    right: Phaser.KeyCode.RIGHT
});

We can query those keys to see whether they are pressed or not by querying the isDown property in the key instances.

When we call the addKeys we need to supply the keycodes for the keys we want to register. These keycodes are stored in the Phaser.KeyCode object, and you can see all the possible keycodes in the documentation.

To control the ship we will add a move method to the Ship sprite class, that will change its body velocity.

To implement all of this, we will need to execute some code every frame. We can achieve this by adding an update method to our game state and add there our custom behaviour.

PlayState.update = function () {
    if (this.keys.left.isDown) {
        this.ship.move(-1);
    }
    // ...
};
