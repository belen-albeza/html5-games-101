Phaser's Arcade physics engine provides collision detection with bounding boxes.

To make the bullets destroy aliens, we need to check for overlapping between these groups of sprites every frame, with the overlap. We will provide a callback to this method to specify what we want to do in case there is a collision.

game.physics.arcade.overlap(bulletGroup, enemyGroup, function (bullet, enemy) {
    enemy.kill();
});
